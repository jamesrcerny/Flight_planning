## Overview

Learning how to calculate the wind correction angle, pressure, and density altitude is a fundamental skill learned early on in flight training. 
This application was made to aid students in cross country flight planning by calculating wind correction angle, pressure, and density altitude.

## Features

- Easy user interface with two simple menu options for selecting between Pressure and Density Altitude Calculator and Wind Angle Calculator.
- Fillable text boxes with descriptions and units to guide the user.
- Calculate button
- Main Menu buttons for returning to the main menu.

## Tech Stack

- **Language:** Python
- **Framework / UI:** CustomTkinter
- **Key libraries:** math
- **Platform:** Windows 10/11 Desktop Application

## Getting Started

### Run the app
1. Go to dist/
2. Download flight_planning_app.exe
3. Run `flight_planning_app.exe`.


## Why I Built This

When I first started learning Python, I came up with the idea of making a flight planning application from scratch. 
All of the formulas I came up with were made from scratch, and I barely touched the internet to figure out the math that went behind the wind correction angle.
I spent hours debugging and problem-solving until I finally found the solution. It wasn't until years later that I decided to make the simple 1-file Python program into a 
fully operational GUI. I refactored the code and organized it into a functions file and a main file that hosted the GUI framework. Once the project was close, I used AI to help
polish the GUI framework.

## Roadmap

- In the future, I plan to develop the program into an IOS application and an Android application.
- I also plan to add more features such as fuel calculations, weight and balance, and the ability to store aircraft profiles that store information about tail numbers into a JSON file for future calculations.
